# CVS PHANToM - Capacitor Android App

GBKK Tracker + Tools รวมเป็น Android APK

## Tech Stack
- Capacitor 6.x
- Leaflet (OpenStreetMap)
- @capacitor/geolocation - Native GPS

## Files
- `www/index.html` - GBKK Tracker (main)
- `www/tools/Report.html` - Report Tool
- `www/tools/Mjob_Helper.html` - Mjob Helper
- `www/tools/Timestamp.html` - Timestamp Tool

## Build
Uses Codemagic for cloud builds (no Android Studio required).
